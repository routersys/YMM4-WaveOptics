YukkuriMovieMaker4（YMM4）上で動作する、**フラウンホーファー回折理論に基づいて計算した点像分布関数（PSF）で映像を畳み込む映像エフェクトプラグイン**です。
波長・F値・画素ピッチ・開口形状・ゼルニケ波面収差係数をパラメータとして設定でき、円形開口・正多角形開口・中央遮蔽・6 種類の波面収差項に対応します。
PSF の計算品質は FFT 格子サイズで切り替え可能で、各パラメータはアニメーションに対応しています。

## 概要

本プラグインは YMM4 の「映像エフェクト」として動作し、エフェクトの種類一覧では「波動光学」として表示されます。

瞳関数を FFT で変換して強度分布（PSF）を計算し、得られた PSF カーネルを Direct2D カスタムピクセルシェーダーで映像に畳み込むことで、回折・収差による像のにじみを物理的に再現します。PSF はエネルギーが保存されるよう正規化され、「利得」パラメータで表示輝度を調整できます。

PSF の計算には単色のフラウンホーファー回折を仮定します。白色光や多色光のシミュレーションには対応していません。

このエフェクトは AviUtl 向けの EXO 出力（`.exo`）には対応していません。

## インストール方法

1. [Releases](https://github.com/routersys/YMM4-WaveOptics/releases/latest) ページから最新のプラグインファイル（`.ymme`）をダウンロードしてください。
2. YMM4 が起動していないことを確認し、ダウンロードしたファイルを実行（ダブルクリック）してインストールします。
3. YMM4 を起動し、タイムライン上のアイテムに映像エフェクトを追加します。
4. 映像エフェクトの種類として **「波動光学」** を選択してください。

## 主な機能

### 1. フラウンホーファー回折によるPSF計算

瞳関数のグリッド上の各サンプルに対してゼルニケ収差で変調した位相を与え、FFT を適用して焦点面の強度分布（PSF）を計算します。PSF は双線形補間でカーネルにサンプリングされ、エネルギーが保存されるよう正規化されます。

計算品質は「品質」パラメータの FFT 格子サイズで切り替えられます。

| 品質 | FFT 格子サイズ |
|---|---|
| 高速 | 128 × 128 |
| 標準 | 256 × 256 |
| 高精度 | 512 × 512 |

瞳関数の直径は FFT 格子サイズの 1/4 です。開口形状は円形と正多角形から選択でき、正多角形では羽根枚数（3〜32）と回転角を設定できます。中央遮蔽（開口半径に対する割合 0〜95%）にも対応します。

### 2. ゼルニケ多項式による波面収差

以下の 6 種類のゼルニケ収差係数を独立して設定できます。各係数の単位は波数（waves）で、設定範囲は -10〜+10 waves です。

| 収差項 | 方向 |
|---|---|
| デフォーカス | — |
| 縦横非点収差 | 0 度 |
| 斜め非点収差 | 45 度 |
| 水平コマ収差 | 水平方向 |
| 垂直コマ収差 | 垂直方向 |
| 球面収差 | — |

### 3. D2D1カスタムシェーダーによる畳み込み

PSF カーネルを R32G32B32A32_Float 形式のビットマップとして Direct2D デバイスコンテキストへ渡し、HLSL ピクセルシェーダーで畳み込みを行います。カーネル最大サイズは 31×31（半径 15 px）です。

適用量が 0% の場合、または出力矩形の計算で半径が 0 になる場合は畳み込みを省略してパススルーとなります。畳み込み後に「利得」を乗算することでピーク輝度を調整できます。

### 4. アニメーション対応パラメータ

適用量・利得・波長・F値・画素ピッチ・開口回転・中央遮蔽・6 種類の収差係数はすべてアニメーション対応であり、キーフレームを設定することで時間に応じて値を変化させることができます。

パラメータの変更を検出した場合のみ PSF を再計算します。

## 制限事項

- カーネル半径は 1 〜 15 px の範囲で指定できます（カーネルサイズ最大 31×31）。
- 波長の設定範囲は 380 〜 780 nm（可視光域）です。
- 計算は単色のフラウンホーファー回折に限定されます。白色光や多色光には対応していません。
- AviUtl 向けの EXO 出力（`.exo`）には対応していません。

---

## 注意事項

- **品質と計算負荷**: 品質「高精度」では FFT 格子サイズが 512 になるため、パラメータ変更時の計算負荷が増加します。リアルタイムプレビュー中にパラメータを頻繁に変更する場合は「高速」への切り替えを検討してください。
- **カーネル半径と出力サイズ**: 適用量が 0% より大きい場合、出力矩形は入力矩形よりもカーネル半径分だけ拡張されます。
- **適用量が 0% の場合**: 畳み込みは実行されず、入力がそのまま出力されます。
- **AviUtl 非対応**: 本プラグインは AviUtl 向けの EXO 出力（`.exo`）に対応していません。
- **本プラグインを使用する前に、YMM4 プロジェクトファイルのバックアップを作成することを推奨します。**
